# railsgirls

A program written to practise using Ruby on Rails.



* Ruby version: 2.4.1

## Technologies used
- Ruby on Rails
- Bootstrap, for styling
- Database
- Devise, for authentication


## Instructions to use the program
- clone this repository into your machine
- run ```bundle install```
- ```rake db:migrate```
- ```rails server```
- visit http://localhost:3000/
